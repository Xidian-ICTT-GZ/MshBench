    //@ requires true;
    //@ ensures true;
    public void foo(int x) {
      ...
    }
- Method contracts and inline proof commands must use line comments, each on its own line in method or class bodies:
  //@ requires ...;
  //@ ensures ...;
  //@ invariant ...;
  //@ open P(...);
  //@ close P(...);
